# Student Course Management System (Flask)

This is a Flask-based web application connected with a MySQL database.

## Features
- Manage students and courses
- Enroll students into courses
- Web interface using Flask templates

## Tech Stack
- Python (Flask)
- MySQL
- HTML, CSS

## How to Run
1. Clone the repository
2. Install dependencies
3. Run `python app.py`
4. Open browser at `http://127.0.0.1:5000`
